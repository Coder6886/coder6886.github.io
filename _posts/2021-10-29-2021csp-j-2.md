# 复赛真题

## 1 分糖果

```c++
#include<bits/stdc++.h>

using namespace std;
int main(){
    int n,l,r;
    cin >> n >> l >> r;
    if(r - l > n)cout << n-1 << endl;
    else{
        if(r / n * n - 1 <= r && r / n * n - 1 >= l)cout <<n-1 <<endl;
        else if(l / n * n - 1 + n >= l && l / n * n - 1 + n <= r)cout<< n-1  <<endl;
        else cout << r%n<< endl;
    }
    return 0;
}
```
