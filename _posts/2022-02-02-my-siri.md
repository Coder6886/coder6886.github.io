        global framesay, picshowsay
        pos = event.Position
        pos = self.ClientToScreen(pos)
        #拖动
        
        if self.leftDown and event.leftIsDown:
            if pos.x!=self.x or pos.y!=self.y:
                self.moving = True
            self.position[0] += pos.x-self.x
            self.position[1] += pos.y-self.y

                framesay.onClose()
            else:
                framesay.Show()
                picshowsay = True
        else:
            self.moving = False
        self.x = pos.x
        self.y = pos.y
        self.leftDown = event.leftIsDown

# 功能

我们这个小Siri叫Lily。

运行后可以看天气和时间，关闭可以关闭图标，但后台语音处理线程不会关闭。

想再次打开图标是说:"hey,lily!"

我这个应用是一个英文应用，大家有兴趣可以把她变成中文应用。

此应用除了天气预报外无需网络。

# 主体思路：

## 	后台：
```

​使用vosk语音识别系统识别语音，枚举所有表达的可能性，并用pyttsx3读出结果。

​这是一个线程。
```

## 	前台：

```
​使用wxpython制作自创lily小图标，这个图标现在有用了，点击就会出现你说的话。

​这是主程序
```

​当我关闭lily时，她不会真正关闭，只会不显示界面，程序没有真正关闭，真正关闭需要用到命令行或任务管理器。全局变量picshow就是界面显示与否的布尔变量。当lily不显示时，程序只会听取"hey,lily!"，当听到"hey,lily!"时，就会显示lily的图标并听取别的数据。
